HLA Type Validate
[24 January 2003]    To print this page, Click here first

The HLA Type Validate is a process which compares the HLA type values on the Mumps system to the types in the Oracle system.

Procedure

  1. Log into the MUMPS system, select option 95;11;9 - HLA Type Validate. Answer the prompts as follows:
      Enter output device [HFS] :[press Enter]
      File name: r:\hla\export\hla_type_validate.txt
      (If the system asks if it is OK to overwrite, enter 'Y')
  2. The system writes a file with all hla types done in the past 30 days. When it is finished, you can log out of Mumps
  3. Log into the Oracle system. Before running the validation, run the bridge program, to ensure all Mumps XML files are applied to Oracle. Then, select menu item Tools|HLA Validate. The window opens - click the [Run Validation] button, then select the file you just created in Mumps
  4. The system checks each row in the file against the HLA type in Oracle. Any differences are displayed in the white area below the button. When the validate is complete, a summary appears near the bottom of the window. If there are no differences, the summary will say:
      'Validation complete - Record count xxx, Error count 0'
  5. The procedure is complete. If there are any errors, page support.
