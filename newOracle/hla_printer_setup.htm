Printer setup

Last revision: Jan 13, 2005   To print this page, Click here first

Printer IP Addresses

  • HP4 Si 10.10.1.207
  • HP 4050 10.10.1.202
  • HP 4300 10.10.1.203
  • HP 4300 Office  10.10.1.204

Adding the HP4050 printer

  1. Select Start|Setting|Printers. Double-click Add Printer, and then click Next.
  2. Click Local printer, clear the Automatically detect my printer check box, and then click Next.
  3. Click Create a new port, click TCP/IP prot, and then click Next.
  4. Select TCP/IP port, enter IP address 10.10.1.202, click OK
  5. When the printer prompts for a driver, browse to the folder p:\drivers\hp4050\newHp4050\. The computer will find the INF file you need and it will complete the installation.

Adding the HP4 Si printer

  1. Select Start|Setting|Printers. Double-click Add Printer, and then click Next.
  2. Select network printer, then click Next.
  3. Select Type printer name, clear the name field, then click next
  4. The window displays a list of shared printers. Select \\Ts1\HPLaserJet 4Si, then click Next
  5. Select YES for Use as default printer, then Next
  6. Click Finish

Adding the HP 4300

  1. Decide which HP4300 printer you want to add: the one in the sequencing area, 10.10.1.203, or the one in the Office, 10.10.1.204
  2. Get the installation CD from the file cabinet in the server cubicle.
  3. Select Start|Setting|Printers. Double-click Add Printer, and then click Next.
  4. Select Local printer, clear the 'Automatically detect...' box, click [Next]
  5. Select Create a new port, then select TCP/IP port from the port type list, click [Next]
  6. The Port Wizard window opens, click [Next], enter printer name 10.10.1.203 (or 10.10.1.204), then click [Next], then [Finish]. Wait for it to finish, about 10 seconds, and redisplay the Add Printer window again
  7. Click [Have Disk], insert the HP4300 installation CD
  8. At the 'Install from Disk' field, browse on the CD (usually drive D:) to folder \English\Drivers\Win2000_XP\PCL6. Click once on the HPxxxx.inf file, click [Open], then [OK], then [Next]
  9. Select YES for Use as default printer, then Next,  Next (do not share), Next (print test page)
  10. Click Finish
