Overview

The HLA Oracle system contains templates which define how test orders are grouped onto worksheets.  A worksheet is a table of related test orders, e.g., Patient A,B,DR test orders for FAMILY MEMBERS.  As the test orders are added, they accumulate on the worksheet. Based on the test order, the test may take up 2,3 or more worksheet slots.
