NMDP LBSW Export

Last revision: January 27, 2005    To print this page, Click here first

The NMDP LBSW Export window allows you to add/remove specimens from the Open batch, create new batches, and export HLA types to the NMDP using the HML or LBSW format. To open this window, select Reports, NMDP Options, NMDP Export from the main menu.

Additions:

·         Version 1.23 – View tab page displays the most recent batch

·         Version 2.05 – Window is now called ‘NMDP Export’, added HML format for export file

·         Version 3.01 – Filter results by HLA locus (27-Jan-05)

·         Version 3.08 – Added TAB_DELIM format for Hong Kong contract

Open batch Maintenance

The system displays the specimens which are currently in the Open Batch. In this section you can add or remove specimens to/from the Open batch, and you can create new batches.

  1. To add a specimen to the Open batch, click the [Add specimen] button, then select a specimen from the lookup window. The system warns you if you select a specimen which is already in the Open Batch. To remove s specimen, click the specimen row first, to select it (you will see it highlighted), then click [Remove specimen]. The system will prompt you to confirm, then it will remove the specimen.
  2. Create a new batch. Select an NMDP contract from the drop-down list and click the [Create] button. The system displays a confirmation prompt which tells you how many specimens will be included in the new batch. Click [OK] to create this batch, or [Cancel] if you do not wish to create it. When you click OK, the qualifying specimens are moved from the Open batch to the new batch.

View and Export

1)     View. When the View tab page opens, it retrieves and displays the most recent NMDP batch. If you want to view another batch instead, click the 'Select another batch' button - the system will open up a selection window.

2)     Include Loci (version 3.01).  The program displays a new LOCI grid above the NMDP batch grid.  In this grid, you can select which HLA loci to include, for the entire batch, or for individual specimens

a)     The first row displayed is labeled ‘DEFAULT’. Check or uncheck the A,B,C,DRB1,DRB345 or DQB locus values on this default row. Changes on this row will apply to every specimen in the list

b)     To use specific settings for one specimen, click on that specimen in the lower (NMDP batch) grid. The LOCI grid displays that specimen.  Check or uncheck the locus values in the row.

c)      When you click the Export button, the program moves the LOCI grid back to default.  CHECK THE DEFAULT VALUES before creating the export.

d)     Note: Each time you open a batch, the LOCI grid shows all values checked.

3)     Export. To create an export file, click the [Export] button. The system prompts you to confirm, then it creates the export. As the file is being created, the status area displays the message 'Creating LBSW Export...', and when it is done is displays the message 'Export is complete'. The file is created in directory 'R:\HLA\MCI'.

4)     For hml files, filename is HML_<batch-number>.xml,  e.g., hml_1234.xml.

5)     For lbsw files, filename is NM1p<batch-number>.prn, e.g. nm1p1234.prn

6)     (Discontinued in version 2.05) Adding XX to allele values. With version 1.35, 2-char molecular values such as A*02 will have 'XX' added to the end for reporting to NMDP. E.g., the value A*02 is reported as A*02XX.

7)     HML Export, version 2.05.  On the View tab page, you now select the HML or LBSW format – HML is the default.  The computer puts HML files in the same folder as LBSW files.  The filename is ‘HML_’ + batch-number.xml. It is OK to create both an hml and lbsw file for the same batch.

1)     TAB_DELIM export, version 3.08.  Select TAB_DELIM format, computer creates file ‘TAB_DELIM_”+batch#+’txt’
