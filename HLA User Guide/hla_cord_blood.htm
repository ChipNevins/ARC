Workflow for cords

  1. Cord Blood program sends cord sample to HLA lab. HLA lab logs in the sample with ISBT label number.
  2. HLA lab performs tests and assigns HLA type, indicates hla type is final, and adds the cord to the NMDP Open Batch
  3. HLA lab exports a file of HLA typings to Cord Blood program. See Cord Export
  4. Cord Blood program sends file of NMDP Ids to HLA lab. HLA lab imports this file. See Cord Import
  5. HLA lab sends LBSW file of cords to the NMDP. See section NMDP Export

Cord Export

  1. When the window opens, select a cord donor site from the drop-down list, and select a typing method. Note: Worcester cords are under the donor site DEDHAM
  2. Click the 'Retrieve Cord List' button to see the list of cords on the computer screen. Default sort order is by last-name, this was the way cord ids were entered in the HLA system. You can click on any column to sort by that column.
  3. Click the 'Export' button to create the export file. The system creates a comma-delimited text file with these fields: ID, ALLLELDT, HLADT, ACTION, A1, A2, B1, B2, DA1, DA2, DB1, DB2, RB11, RB12, RB31, RB32, RB41, RB42, RB51, RB52, METHOD, HLA_REF. The A1,A2,B1,B2 are the serologic equivalent values of the A/B antigens. The value HLA_REF is the identifiers assigned to the sample by the HLA lab.
  4. ZIP this file and email it to the data manager of the cord site.

Cord Import

  1. When the site emails a file of NMDP Ids, copy the file to a diskette, or to folder r:\hla\cord\imports.
  2. In the HLA application, select menu Data Entry |Cord Import
  3. Click the '...' button to select the file you want to import.
  4. Indicate which type of file you are importing. Right now, the only choice is St Paul NMDP Ids. The computer displays a description of the file format required - this would be used only in trouble-shooting.
  5. Click the 'Import' button. The computer imports the NMDP IDs and displays its progress in the log file display area.
  6. When the process is done, the log file area will say that import is complete. You can close the window.
