LBSW Filter

Last revision: July 10, 2003    To print this page, Click here first

The LBSW filter window is used to convert the HLA reference numbers in LBSW files to NMDP identifiers. It can also change the lab code for every record in the file.

Note: You can also use this window just to view LBSW files

Procedure

  1. In the HLA Oracle system, select Tools|LBSW Filter. The system opens the LBSW Filter window.
  2. First, select a file to filter. This file must be an LBSW file created in the HLA system or Quicktype system. Click on the [Select] button, then move to the folder which contains your file. The system displays files of type .TXT and .PRN. Select the file you want.
  3. The system displays the records of the file in the text area below the selection button. It leaves some space to the left of each record. The system reads the HLA ref number from each record, and looks up the NMDP Id for this record. if no NMDP Id is found, the system displays a red 'x' next to the line.
  4. You can click on any record in the file - the system displays the fomatted contents for each filed on the right.
  5. To update the file to include NMDP Ids, click the [ ] NMDP Id checkbox.
  6. To convert NMDP Ids back the Ref#, click the [ ] Add R-Numbers checkbox.
  7. To change the lab code for every record, enter the lab code and click the [ ] Lab Code checkbox
  8. Click the Build file button. Note that the system will not build a file if any records were marked as 'x'
  9. The system creates a new, filtered file with the name 'new' prefixed to the original file name, in the same folder.
  10. You can view the formatted fields of the new file by clicking on a record. Check the new file by viewing the formatted data for several records
