The inventory functions allow the user to set up box locations, add specimens to defined boxes, list the contents of selected boxes, display the inventory for a specimen. An audit trail is kept of all updates to the inventory locations of specimens.

Inventory - Add Specimens

This window lets you add inventory items for specimens, individually or in batches. Note: A single specimen may have multiple inventory items (locations).

Adding/updaing individual specimens

  1. Select Data Entry|Inventory from the main menu. The computer opens the inventory window. This window has buttons at the top and a large data entry grid below.
  2. Press the [Select Specimen] button, and select a specimen from the lookup window. The computer displays the Pt/Donor name, accession date,and any current inventory for this specimen.
    • If this specimen has never been added to Inventory, the computer will create a new, blank row in the data entry section.
    • If the specimen has current inventory, no row is added. You must either select this row to change it, or check off the [ ] new item checkbox to add a new inventory item.
  3. Enter values for Box number, contents, items added/removed, and comment.
  4. Click [Select Specimen] to continue entering items. When you are done, click [Save Changes]
  5. If you put an item in the wrong box, you must correct it by first changing the amount to zero for this item, and then creating a new item to add the specimen to the correct box. You cannot update the BOX number field - it is part of the record key.

Other buttons on this window

  • New Box. Opens the Box Entry window so that you can add a new box without leaving the Inventory Window.
  • Cancel changes. Cancels all changes since the last Save action. (Note: this does not cancel Box Entry changes you made)
  • Delete Row. Removes a row (new row ONLY!) from the data entry grid.

    Adding R-Number range

    1. Click the [Refnum] button. When the computer opens the Select R# Range window, follow these steps:
      • Enter starting/ending R# values and click [Retrieve]. The computer displays the detail data for each specimen in the range,and a count.
      • If these are the specimens you want, click OK. The computer inserts each specimen into the data entry grid.
    2. Enter Box#, contents, Items added and comment for the first row. To copy the values to the rows below, click the value you entered in row 1, then click the [Copy value ...] button. Do this for each column.
    3. When you are done with changes for the R# range, click [Save Changes]

    Add from packing List

    1. Click the [PList] button. When the computer opens the Select Packing List window, follow these steps:
      • Select the packing list you want. The computer displays the line items of this list.
      • Select the rows you want, then click [OK]. The computer inserts each specimen into the data entry grid.
    2. Add Box#, contents, items added and comments as described above.

    Update entries in Box

    1. Click the [Box] button. When the computer opens the Select Box List window, follow these steps:
      • Select the Box you want. The computer displays the Box inventory.
      • Click [OK]. The computer asks if you want to empty the box. If you answer YES, the computer adds each box item to the data entry grid with the Items to Remove set. If you answer NO, the computer adds each box item to the data entry grid with no added/removed values set.
    2. Update the added/removed values, then click [Save Changes]
    3. If you also want to delete the Box, you must do this through the Box Dictionary.

    Box Entry

    1. Select DataEntry | Box Entry from the main menu. The computer opens a window with the Box List on the left side.

    2. To enter a NEW box, click the [NEW] button, enter the data values, then click [Save]. The computer saves the entry and adds it to the Box List.
    3. The data values you enter are:
      • Stack# - A numeric value for the stack
      • Label in stack - the value A,B,C, etc. or some other alpha value
      • Max items - The capacity of the box, usually 81
      • Contents - A short description of the type of item stored in the box
      • Description - A longer description - free text
    4. To enter a new box which copies the values from a previus box, select the previous box from the Box List, then click [DUPLICATE]. The computern opens a new data entry area with the values copied from this box. You must enter a new Label-In-Stack value. Click [Save] when done.

    Inventory Report

    1. Select menu item Reports | Inventory report. The computer opens a window with a Box list on the left and a detail window on the right.
    2. Select the boxes you want to display, and click [Retrieve]. The data is displayed in the detail window, sorted by Box number.
