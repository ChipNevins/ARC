Person Inquiry Window

Last revision: march 19, 2004    To print this page, Click here first

The Person Inquiry window is used to display and print the information for a single patient or donor.

Additions

Version 1.22 - Family-relation comment added to display
Version 1.43 - Added Inventory section

Version 2.01 – Added Panels section

Procedure

  1. Select menu option Tools|Inquiry. The Person Inquiry opens,and it immediately opens the Person Lookup window. Use the Person Lookup window to select a patient or donor
  2. The system retrieves the data for the donor and displays it. You can scroll up/down and left/right in this display to see all the data.
  3. To print the data, click the printer icon in the toolbar, or press ctrl-P.
  4. You can click the LOOKUP button to select another patient or donor.
