MAC Network

Last revision: January 23, 2004    To print this page, Click here first

This page describes startup, login and shutdown procedures for the Mac machines in the HLA lab.

Introduction

There are now four Mac machines in the HLA lab which are used to run HLA Analysis software. The machine names are Mac1, Mac2, Mac3, Mac4, and they all connected to the local area network (when the network cable is plugged in). This means that each Mac can copy and use files from the TS1 server or from the other Macs. The Macs all run MAC OS 9.2 as their operating system. The sequencing applications do not yet run on MAC OS X.

Connecting to other Macs

Suppose you are on Mac2 and you want to connect to Mac1. Follow these steps:
  • Open Chooser (under the Apple menu), then select Appleshare.
  • The computer opens a window which displays icons for other Macs – select Mac1.
  • The computer displays a login window, enter username=Mac2 pw=***. (The username is the name of the Mac you are on.)
  • The computer opens another window, letting you select the Mac1 HD – simply click and an icon named Mac1 HD will appear on your desktop.
  • Note: The password is the same to connect to any of the Macs.

Connecting to TS1

  • Open Chooser (under the Apple menu)
  • Select Appleshare, then select TS1
  • The computer displays a login window, and you enter these values: username=hlauser, pw=****

Startup and Shutdown

  • Start-up. Press the button on the front of the Mac system. If you start-up a Mac with no network connection, you can always connect a network cable to it later, without restarting. We did not set these connections to be established automatically at start-up, because sometimes the Mac will not be connected to the network, e.g., Mac3 shares a single cable with a PC.
  • Shutdown: Select menu item Special|Shutdown. The system shuts down and powers off.

Disk Doctor

  • The Norton Utilities for Mac CD is in the file cabinet in the server cubicle. This CD contains Disk Doctor and Optimizer
  • In January 2004, Chip ran Disk Doctor and Optimizer on Mac1, Mac2, Mac3 and Mac4. It ran fil eon Mac1 and Mac 4, but did not work cleanly on Mac2 and Mac3. However, it did optimize the files on all the Macs.
  • To use Disk Doctor, insert the CD and reboot the Mac from the CD (Hold down the Option key while restarting).

Applications on the Mac

  • File conversion: The Sample File Win to Mac program is on the desktop of each Mac. Double-click it, then, select the files or folders you want to convert.
  • Sequencing applications: All the machines have these installed apps: Sequencing Analysis, Matchmaker, Match Tools, MT Navigator.
