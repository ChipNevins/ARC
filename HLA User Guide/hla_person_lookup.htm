Person Lookup Window

Last revision: March 26, 2004    To print this page, Click here first

The Person Lookup window is used to select a patient or donor for a data entry or reporting task.

Procedure

  1. When the Lookup window opens, enter the identifier or name of the patient or donor. You may enter a Ref#, DRN or Name (full or partial) as a lookup value. After entering the name, press the TAB key
  2. The system looks up your value and displays a list of matching records. You can move up and down through this list, with the mouse or with the keyboard. As you move through the list, the system displays the Ref# and Name of the current record, in a large font below the list. If only one record matches your lookup value, that record is displayed.
  3. Press ENTER or click the Select button to select the current record and close the lookup window. To exit without selecting a record, click the 'X' close icon at the top right of the window.
  4. If you entered an incorrect lookup value, e.g. SMUTH instead of SMITH, you can move back to the lookup field and enter a new value.
  5. If you enter name as LAST, FIRST MI the computer ignores the middle initial when it reports matches. With this change, if you enter the name DOE, JANE T., the computer will show all Jane Doe records (not just Jane T. Doe records), and you can check if this pt/donor is really new, or was previously logged in without a middle initial.

Recent List
