Instructions to Install HLA SqlServer Client.txt


First time install on a PC:
-------------------------------------------------------------- 
Follow this instruction on a new machine. It loads common files and
the most recent version of HlaSqlServer

1. Log onto the PC under the HLAUSER account.
2. Start > MyComputer, then navigate to folder p:\HlaSqlserver\XcopyDeploy
3. Double click the batch file 'Install Runtime.bat'
4. Double click the batch file 'Copy Latest Version.bat'
5. Copy the HLA SqlServer shortcut (the 'globe') to the desktop of the PC


To upload a new version once I have made it available
------------------------------------------------------
1. Start > Run
2. Click browse, then navigate to folder p:\HlaSqlServer\XCopyDeploy
3. Double click the file 'Copy Latest Version.bat'
